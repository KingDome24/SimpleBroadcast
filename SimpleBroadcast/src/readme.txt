Hello dear SimpleBroadcast user,

thank you that you're using my plugin.
In this "readme.txt" you propably find some helpful information if you're having problems.

First of all please be sure that the prefix, suffix and the messages
are standing inside of single quotes ('...') so that the colour codes and also the variables can work properly.
If you want to use a single quote (') please don't write it out.
So that plugin still works you have to use a variable (%sq%) instead.
Finally I recommend you to not ignore the update messages if they're shown
in the log or ingame. An update is always good because new features which you
propably like could be added or annoying bugs could be fixed.

Yeah, that's all what I wanted to say. Thank you for reading this text
and if you're still having problems don't hesitate to ask me on
the BukkitDev page.
